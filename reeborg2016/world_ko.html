도움말

        
도움이 필요하시다면, 이 문서를 보시길 바랍니다.
        파이썬을 사용하신다면, help() 코드를 실행해서 명령어 목록을 보세요 아니면, help(move)을 실행해서
         move() 함수에 대한 도움말, 등을 얻으세요.

  

  

     기존 월드를 수정하여 자신 만의 월드를 만들 수 있습니다.
  




  

    
월드 : 창조, 수정..

       기존 월드를 수정하여 자신 만의 월드를 만들 수 있습니다.
  

  

    
    완성된 내용을 저장하려면, 퍼머링크 버튼을 클릭하세요.
  복사되어 저장된 위치를 골라서 퍼머링크를 만들 수 있습니다.
  


  

    
    파일의 퍼머링크 복사본을 저장합니다. 자신의 월드를 만들때 유용합니다.
  


  

    
    (json 확장자로) 월드를 컴퓨터에 저장합니다.
  

  

    
    컴퓨터안의 파일로 월드를 불러옵니다.
  


  

     브라우저 메모리 월드에 현재 상태의 복사본을 저장합니다.
     선택한 이름으로(로컬 저장소) 유효하게 사용할 수 있도록 기존 월드를 열 수 있습니다.
   


  

  
버튼을 클릭하면 브라우져의 메모리에 저장된 월드를 제거합니다:

  


  

  
에디터

  

    
     에디터의 소스코드를 저장합니다.
  


  

    
     로컬 저장소를 열고 소스코드를 불러옵니다.
  






미래에 생겨날 기능

  

    
  


  

    
     흔적을 리보그만 남기고 지웁니다. 이 기능은 프로그램이 멈추었을때 유용합니다.
  




  

    
     다른 사용자는 Mozilla의 TogetherJS를 이용하여 협업에 참여 할 수 있습니다.

리보그에 대해..


    
Inspired by Guido van Robot, itself inspired by the original Karel the Robot
    created by Richard Pattis, Reeborg first appeared in RUR-PLE, a program I created
    to learn Python, picking up programming as a hobby.


    
 I hope you enjoy learning computer programming with Reeborg.

    
Feel free to contact me at
    Andre.Roberge@gmail.com


    
Reeborg is free and will remain free to use and to share under a
    
    Creative Commons License.
    If you are using it to teach programming to students,
    please contact me to let me know about it and so that I can keep you informed of major changes.
    You might also want to make a copy of this site on a local server.
